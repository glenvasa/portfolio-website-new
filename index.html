About me


Learning new coding concepts and techniques, through building projects (front-end, back-end, or full-stack) has become a daily passion.

I love the creativity that front-end development allows in displaying impressive, effective applications from the client perspective. The ability to make something for the user that is both visually enticing while useful in its purpose, is compelling.

However, equally, I have developed an appreciation for the back-end business logic that drives a web application. Running a server, connecting a database, and hooking up the the front-end to form a full-stack project, is extremely rewarding.

I'm excited to bring my coding/web development passion, desire to learn, and years of experience as an attorney and business owner to the table for the benefit of your company.

myFlix - React

Project Brief:

Client-side application developed with React and Redux.
Utilizes MyFlix REST API hosted at myflix2020.herokuapp.com.
Server-side code is found at github.com/glenvasa/movie_api.
MongoDB stores User and Movie Collections.

Users Register/Login from Welcome Screen and brought to MainView page that displays all movies in database.
Users have the option to search for movies in MainView or click button to view specific movie's details.
MovieView page contains movie's description, genre name, and director; also allows user to add movie to list of favorites and/or click button to view additional genre or director information.
Profile page contains User's Favorite Movies List in which User can navigate to a specific favorite movie's MovieView page, remove movie from Favorites List, or navigate to Update Profile Page if they wish to modify any login credentials.

Single-Page React Application (SPA).
React Routing used to navigate between views and share URLs.
Built with Parcel.
Uses Bootstrap UI library for styling and responsiveness.
Utilizes a mix of class and functional React components.

Project Info

  • Date - 2020
  • Tools - mongodb, express, react, node, bootstrap, postman
  • Web - myflix2020.netlify.app
  • Project code - github.com/glenvasa/myFlix-client

jibjab

Project Brief:

-Users enter name and select background color in Start Screen before joining the chat.

-Chat Screen displays the conversation, as well as an input field and submit button.

-Provides users with two additional message features: sending images (either from image library or device's camera) and location data displayed as a custom map view.

-Messages, including sent images and location data, are stored in Google Firebase Cloud Storage.

-Users must grant permission to the app prior to accessing image library, camera and location data.

-When offline, the app will display a user's existing messages, utilizing asyncStorage.

Project Info

  • Date - 2021
  • Tools - React Native, Expo Client, Gifted Chat, Google Firestore Database
  • Developed on Android Studio (android) and Expo Go (iphone)
  • Project code - github.com/glenvasa/jibjab
